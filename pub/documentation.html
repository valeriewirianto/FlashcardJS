API Documentation

Getting Started

To use Flashcard.js, several scripts and links must be added to your HTML file. Include the flashcard.js file in the script tag of your HTML file. Also link styling.css to your HTML file. Include jquery library in a script tag. Finally, include your javascript file which will be using Flashcard.js.

<link rel="stylesheet" type="text/css" href='/path/to/styling.css'>
<script defer src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js">
<script defer type="text/javascript" src='/path/to/flashcard.js'>
<script defer type="text/javascript" src='/path/to/examples.js'>


API: Object Creation

To create a single sided flashcard deck, instantiate it using 'FlashcardsSingleSided' and pass the title of the deck as an argument:

const f = new FlashcardsSingleSided("Title of this single sided deck")


To create a double sided flashcard deck, instantiate it using 'FlashcardsDoubleSided' and pass the title of the deck as an argument:

const f2 = new FlashcardsDoubleSided("Title of this double sided deck")


To create an input form that allows users to create their own decks, instantiate it using "InputForm" and pass in a title as an argument:

const form = new InputForm("Title of this input form")


API: Methods

Single sided deck methods

Below is a list of methods available for FlashcardsSingleSided

Method Description Arguments Return Values Example
.getAllOriginalCards() Get all cards in the original deck - Array of single sided card objects, each with a 'front' field for the value on that card f.getAllOriginalCards()
.getAllFavoritedCards() Get all cards in the favorited deck(deck with favorited cards) - Array of single sided card objects, each with a 'front' field for the value on that card f.getAllFavoritedCards()
.addSingleSidedCard(cardValue) Add a single sided card to the deck. Added to the main, original deck, not the favorited deck. String - f.addSingleSidedCard('I am a new card')
.removeSingleSidedCard(cardValue) Removes the card with cardValue as its value. Removes from both main deck and favorites deck. String - f.removeSingleSidedCard('I am a new card')
.addFavorite(cardValue) Adds a card to the favorited cards deck. String - f.addFavorite('I am a favorited card')
.shuffleDeck() Shuffles the deck to a random order. - - f.shuffleDeck()
.disablePreview() Disables the side preview. Removes the panel which displays all the card values at once. - - f.disablePreview()
.selectAnimation(animationName) Selects an animation type for this deck. If this method is not called, the default is no animations. String of one of the following animations:
  • move
  • fade
  • move-and-fade
  • spin
  • roll-blur
  • grow
 - f.selectAnimation('move-and-fade')

Double sided deck methods

Below is a list of methods available for FlashcardsDoubleSided

Method Description Arguments Return Values Example
.getAllOriginalCards() Get all cards in the original deck - Array of double sided card objects, each with a 'front' field, 'back' field, and 'points' field. f2.getAllOriginalCards()
.getAllFavoritedCards() Get all cards in the favorited deck(deck with favorited cards) - Array of double sided card objects, each with a 'front' field, 'back' field, and 'points' field. f2.getAllFavoritedCards()
.addDoubleSidedCard(frontCardValue, backCardValue, points) Adds a card to the main, original deck. String, String, int - f2.addDoubleSidedCard('Front of favorited card', 'Back of favorited card', 20)
.removeDoubleSidedCard(frontCardValue, backCardValue) Removes a card from both the original deck and favorited deck. String, String - f2.removeDoubleSidedCard('Front of favorited card', 'Back of favorited card')
.addFavorite(frontCardValue, backCardValue, points) Adds a card to the favorited cards deck. String, String, int - f2.addFavorite('Front of favorited card', 'Back of favorited card', 20)
.shuffleDeck() Shuffles the deck to a random order. - - f2.shuffleDeck()
.disablePreview() Disables the side preview. Removes the panel which displays all the card values at once. - - f2.disablePreview()
.selectAnimation(animationName) Selects an animation type for this deck. If this method is not called, the default is no animations. String of one of the following animations:
  • move
  • fade
  • move-and-fade
  • spin
  • roll-blur
  • grow
 - f2.selectAnimation('move-and-fade')
.disableSkip() Disables the skip feature, i.e. users will not be able to skip answering a card's question. - - f2.disableSkip()
.disableLeaderboard() Prevents the leaderboard from being displayed. - - f2.disableLeaderboard()
.sortLeaderboardBy(sortBy) Sorts the leaderboard by either top points or top number of questions answered correctly. If this method is never called, the default sort is by points. String that is one of:
  • points
  • numCorrect
 - f2.sortLeaderboardBy('numCorrect')
